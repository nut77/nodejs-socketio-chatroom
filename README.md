# 前言
初学nodejs，实现的一个简单的在线多人聊天系统。
# 技术栈
nodejs express Socket.IO less es6
# 项目运行
**注意：项目是基于nodejs运行，需要提前安装好nodejs环境。**

```
git clone 项目地址
cd nodejs-socketio-chatroom
npm install
node server.js
```
然后访问```localhost:2000/index```即可进入聊天窗口页面
